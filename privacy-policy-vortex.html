Privacy Policy

Effective date: March 18, 2026

Vortex Emulator ("the App") is developed by JosuSM. This page informs you of our policies regarding the collection, use, and disclosure of personal data when you use our App.

1. Data Collection

Vortex Emulator does not collect, store, or transmit any personal data. The App runs entirely on your device.

  • No accounts or registration are required.
  • No analytics, tracking, or advertising SDKs are included.
  • No crash reporting services are integrated.
  • No personal information is collected, shared, or sold.

2. Supported Platforms

The App supports 25+ retro gaming platforms through libretro cores and standalone emulator integrations, including NES, SNES, N64, Game Boy, GBA, NDS, PSX, PSP, PS2, Dreamcast, Saturn, GameCube, Wii, 3DS, Arcade, and others. Core libraries are downloaded on demand and cached locally on your device.

3. Local Storage

The App stores data locally on your device for functionality purposes:

  • Game library database — ROM paths, platform metadata, cover art references, and playtime statistics.
  • ROM files you choose to open.
  • Save files and save states for your games (up to 10 quick-save slots per game).
  • Rewind buffer — temporary in-memory state snapshots for the rewind feature.
  • System files (BIOS) required by certain emulation cores.
  • App preferences — display settings, control layouts, and per-core configurations.
  • Emulation core libraries downloaded from official libretro build servers.
  • ROM patch backups — original ROM files are automatically backed up before applying IPS, UPS, BPS, xdelta, PPF, or APS patches.

All of this data stays on your device and is never transmitted externally.

4. Permissions

The App may request the following permissions:

  • Storage access — to read ROM files and write save data. On Android 11+, broad file access may be requested to navigate your ROM library.
  • Internet — to download emulation cores, standalone emulator APKs, and for multiplayer features.
  • Network state — to detect offline status before attempting downloads.
  • Install packages — to install standalone emulator APKs downloaded from official GitHub repositories (e.g., PPSSPP, Dolphin, Lime3DS, DuckStation, Flycast).
  • Foreground service — to maintain audio playback during background emulation.
  • Vibration — for haptic feedback on touch controls.
  • Wake lock — to keep the screen on during gameplay.

No permission is used to collect or transmit personal data.

5. Internet Access

The App uses internet access exclusively for:

  • Downloading libretro cores from the official libretro build server on first use, then cached locally.
  • Downloading standalone emulators from their official GitHub Releases pages (via the GitHub API).
  • Multiplayer lobby and netplay — connecting to the Vortex lobby server for room creation, matchmaking, and encrypted input relay (see Section 6).

No personal data, device identifiers, or telemetry is transmitted during any of these operations. The App operates fully offline once cores are cached.

6. Multiplayer & Netplay

The App provides online multiplayer through a WebSocket-based lobby server. When using this feature:

  • A random anonymous player ID is generated locally for each session — no login, account, or personal identifier is used.
  • Game input data exchanged between players is end-to-end encrypted using AES-256-GCM. The relay server cannot read game data (zero-knowledge design).
  • Rooms can be password-protected and support 2–8 players with optional region filtering and in-lobby chat.
  • No personal information, device identifiers, or location data is shared with other players or the server.
  • Multiplayer sessions are temporary — no data is retained after disconnection.

7. Standalone Emulator Integration

The App can launch games in standalone emulator apps installed on your device (e.g., PPSSPP, Dolphin, DuckStation, Lime3DS, DraStic, Flycast, Vita3K, RetroArch, and others). When you choose to install a standalone emulator through the App, the APK is downloaded directly from the emulator's official GitHub repository. The App does not modify, wrap, or intercept any data handled by standalone emulators — each runs under its own privacy policy.

8. Third-Party Services

Vortex Emulator does not integrate any third-party services that collect user data. Specifically:

  • No advertising SDKs.
  • No analytics or telemetry services.
  • No crash reporting services (e.g., Firebase, Sentry).
  • No Google Play services dependency.

The only external connections are to the libretro build server (core downloads), the GitHub API (standalone emulator downloads), and the self-hosted Vortex lobby server (multiplayer).

9. Children's Privacy

The App does not knowingly collect any information from children. As the App collects no personal data from any user, it is compliant with COPPA and similar regulations.

10. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated effective date.
